HERITAGE LUXURY JOURNEY

【概要】
Heritage Luxury Journey は、 「歴史と文化を深く味わう、大人のためのラグジュアリーツアー」をテーマにした 架空の旅行ブランドのコーポレートサイトです。
40〜70代を想定ユーザーとし、 上質さ・安心感・操作のしやすさを重視して設計しました。

【制作目的】
⚫︎デザインと実装を一貫して行えることの証明
⚫︎「なぜそうしたか」を説明できるUI / UX設計力の可視化

【URL】
https://colorful-flower.github.io/HERITAGE-LUXURY-JOURNEY/


【想定ユーザー】
⚫︎年齢層：40代〜70代
⚫︎海外旅行経験があり、価格よりも 質・安心感・余裕 を重視する層
⚫︎スマートフォン閲覧が中心だが、PC利用も想定


【使用技術】
　⚫︎HTML
　⚫︎CSS
   　CSS変数によるカラーパレット管理
  　 レスポンシブ対応（PC / SP）
　⚫︎JavaScript
     ハンバーガーメニュー制御
     スクロール制御

【工夫したポイント】
1. ターゲットを意識したUI設計
⚫︎落ち着いた配色・余白設計
⚫︎小さすぎない文字サイズ
⚫︎「派手さ」よりも「信頼感」を優先

2. ヒーローエリアでの第一印象設計
⚫︎ズームアニメーション＋グラデーションオーバーレイ
⚫︎コピーは「誰向けのサイトか」が一瞬で伝わる内容に設計

3. カラーパレットの一元管理
　CSS変数で色指定を整理し、保守性を向上

　:root {
  --color-primary: #1F2A44;
  --color-secondary: #681F2A;
  --color-accent: #C9A85B;
  --color-bg-base: #FAF7F2;
　}

4. ローディング演出（PCのみ）
⚫︎初回アクセス時のみ表示
⚫︎SPではUX低下を防ぐため非表示

【今後の改善予定】
⚫︎サンクスページの実装
⚫︎アクセシビリティ対応（focus制御・コントラスト）
⚫︎CSS構成の整理（共通化）

【最後に】
見た目だけでなく、「なぜこのUIにしたのか」を言語化しながら制作することを意識しました。 
小さな違和感を改善するプロセスを大切に日々スキルアップに励んでいます。
